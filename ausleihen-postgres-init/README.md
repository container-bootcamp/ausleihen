# Docker Container

Erstellt / Initialisert für den ausleihen-scs eine Datenbank in der Postgres DB und legt die benötigten Tabellen an.

## Umgebungsvariablen

### Erfoderlich

* POSTGRES_DB_LOCALE locale muss im OS installiert sein.
* POSTGRES_HOST
* POSTGRES_SUPERUSER
* POSTGRES_SUPERUSER_PASSWORD
* POSTGRES_DB
